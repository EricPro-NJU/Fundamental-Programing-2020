# 迭代求平方根

### 题目背景

本题满分10分

### 题目描述

对于一个正数$a$，计算$a$的三次根号的值。

已知牛顿迭代公式
$$
x_{n+1}=\frac{1}{3}(2x_n+\frac{a}{x_n^2})
$$
当
$$
|x_{n+1}-x_n|<\epsilon
$$
时，$x_{n+1}$即为所求$\sqrt[3]{a}$的值，本题$\epsilon$取1e-7，迭代数组的初值可以为任意值，比如$a$自身。

### 输入格式

一个正数$a$。

### 输出格式

$a$的立方根，**保留2位小数**。

### 说明

#### 输入数据规模

$1<=a<=10^3$
