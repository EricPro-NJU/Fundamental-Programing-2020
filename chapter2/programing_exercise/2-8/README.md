# 完数

### 题目背景

本题满分10分

### 题目描述

我们称这样的数为**完数（Perfect number）**：

若一个正整数的所有真因子（即除自己本身以外的因子）之和恰好等于本身，则称这个正整数为**完数**。

例如：6是完数，它的真因子有1, 2, 3，且$1+2+3=6$.

请你求出$2$到$10,000$**之间的所有完数**。

### 输入格式

本题没有输入数据

### 输出格式

请输出所有的完数，一行输出一个数字。输出格式例如：

```
100
101
102
...
```
