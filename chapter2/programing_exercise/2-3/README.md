# 二进制转换

### 题目背景

本题满分10分

### 题目描述

对于一个十进制正整数$n$，请将其转换为二进制输出。

### 输入格式

一个正整数$n$。

### 输出格式

一行以二进制形式显示的数据，行末无多于换行、空格。

输出的二进制数需要满足：**最高位一定为1**。

### 样例输入输出

#### input1

```
14
```

#### output1

```
1110
```

#### input2

```
64
```

#### output2

```
1000000
```

### 说明

#### 数据规模约定

$1<=n<=10^9$
