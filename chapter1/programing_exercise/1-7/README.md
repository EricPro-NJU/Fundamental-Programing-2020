### 题目描述

小蓝鲸的程序设计基础课上有n个学生，给定每一个学生的期末考试成绩$a_i$，求这n个同学的期末考试总分。

### 输入格式

两行，第一行为一个正整数n代表个数。

第二行为n个非负整数$a_i$，用空格隔开。

### 输出格式

一个非负整数，表示n个学生期末考试总分。

### 输入输出样例

#### input

```
5
88 89 90 91 92
```

#### output

```
450
```

### 说明

#### 数据规模约定

$0<n<=10000$

$0<=a_i<=100$
