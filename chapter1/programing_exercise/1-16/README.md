# Sans的日历

### 题目背景

本题满分10分

Sans在2020年1月1日的时候买了一个今年的日历挂在墙上，但是由于Sans十分懒惰，他每天都忘记去翻日历，导致墙上的日历每天都翻在1月1日那天，从来没有动过。某一天，Sans想知道今天是星期几，于是他找到你帮忙。

### 题目描述

已知2020年1月1日是星期三，给定2020年的一天m月d日，求指定天数的星期数。

### 输入格式

两个正整数，表示某一天的月数和日数。空格隔开

### 输出格式

一行字符串，指示这一天是星期几。

请按规定输出相应的星期：

星期天 输出`Sunday`

星期一 输出`Monday`

星期二 输出`Tuesday`

星期三 输出`Wednesday`

星期四 输出`Thursday`

星期五 输出`Friday`

星期六 输出`Saturday`

### 样例输入输出

#### 输入1

```
10 20
```

#### 输出1

```
Tuesday
```

#### 输入2

```
12 31
```

#### 输出2

````
Thursday
````

### 提示

**2020年是闰年**
