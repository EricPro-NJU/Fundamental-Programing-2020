# 金字塔

### 题目背景

其实用循环可以做很多有意思的事情，比如画画^_^

本题满分5分。

### 题目描述

根据输入的正整数n，输出一个如下的n层图形：

```
      *             1层
     ***            2层
    *****           3层
     ...
*****...*****       n层
```

要求：

1. 第$k$层有$2k-1$个`*`；
2. 第$k+1$层最中间的`*`必须和第$k$层最中间的`*`对齐；
3. 每行的`*`之间均无空格
4. 最后一行第一个`*`前不能有空格。

### 输入格式

一个正整数，代表金字塔图形的层数。

### 输出格式

按要求输出金字塔图形。

#### 输入输出样例

#### 输入1

```
3
```

#### 输出1

```
  *
 ***
*****
```

#### 输入2

```
5
```

#### 输出2

```
    *
   ***
  *****
 *******
*********
```

### 数据规模约定

$0<n<=100$
